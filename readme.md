# Security Server

## Installation
Get an API Sendgrid key and setup your account. Then Add sendgrid key in .env file
and email. Variables are SENDGRID_API_KEY and SENDGRID_EMAIL
```bash
$ npm install
$ docker-compose up --build
```

## Running the app

```bash
# docker-compose up
```

## Swagger

http://localhost:3000/swagger
